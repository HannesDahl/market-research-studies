# QQQ Consecutive Candle Study

## Objective
Analyze continuation and reversal behavior after consecutive same-colored 5-minute candles in QQQ.

## Segmentation
- n=4 runs
- n=5 runs
- Morning session
- Midday session
- Green runs
- Red runs

## Metrics
- Continuation probability
- Reversal probability
- Target hit rates
- Session-based behavior

## Key Findings
- Small continuation edge after 4–5 candle runs
- Stronger continuation during morning session
- Edge weakens during midday
- Larger targets reduce continuation expectancy
